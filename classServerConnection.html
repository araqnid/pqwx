Detailed Description

Defines connection parameters for a server.

This now-horribly-misnamed class contains the requisite information to make an initial connection to a server, including a suggested "administrative" database name. It may also be updated with some discovered attributes after connecting, for example to note if a password is required to connect, server version or server-level capabilities.
