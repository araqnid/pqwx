Detailed Description

Callback interface for non-modal operation of the connection dialogue.

When used non-modally, an callback implementing this interface is supplied to be notified when the connection is established, or if the user gives up and cancels.

Implementations must be allocated on the heap, as they will be deleted just after one of the callback methods is called.
